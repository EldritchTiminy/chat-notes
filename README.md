# Notes - JavaScript Practice
A simple note-taking web app. Made to practice JavaScript.

## Table of Contents
- [Features](#features)
- [Usage](#usage)
- [Tech Stack](#tech-stack)
- [Roadmap](#roadmap)

## Features
- Uses JavaScript to write and delete notes. Stores notes using the WebStorage API.

## Usage
- Type a note that you wish to save in the input field and click the "Save Note" button.
- Clear all notes by pressing the "Clear All Notes" button.

## Tech Stack
- **HTML5**
- **JavaScript (ES6+)**
